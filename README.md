# UFC Fight Prediction Quarto Website

This is a Quarto website version of the STAT2005 UFC modelling project.

## Local preview

```bash
quarto preview
```

## Render for GitHub Pages

```bash
quarto render
```

The rendered site is written to `docs/`, which can be published through GitHub Pages.

## Scroll frames

Frame images live in `assets/frames/`:

- `frame_001.jpg`
- `frame_002.jpg`

More frames can be added later using the same naming pattern and updating `scroll-frames.js`.
